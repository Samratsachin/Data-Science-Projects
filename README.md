## 📌 Overview  
**Climate Change Modelling** is an **NLP-powered sentiment analysis tool** that predicts public opinion about climate-related activities.  
Using text data from **NASA**, **NOAA**, **climate agencies**, and **social media**, it classifies a statement as **Positive**, **Negative**, or **Neutral**.  

🔹 Built with **Python** and **Flask**  
🔹 Uses **real-world climate datasets**  
🔹 Web app for instant predictions

## 📊 Data Sources  
- 🌡 **NASA** — Climate reports, global temperature anomalies  
- 🌍 **NOAA** — Weather summaries, climate change bulletins  
- 📄 **Climate Agencies** — Research articles & official statements  
- 💬 **Facebook Pages** — Public posts and discussions on climate topics  

---

## 🛠 Tech Stack  

| Category           | Technologies Used |
|--------------------|-------------------|
| **Programming**    | Python (Pandas, NumPy) |
| **NLP**            | NLTK, spaCy, TextBlob |
| **Machine Learning**| Scikit-learn (TF-IDF, Logistic Regression, Naive Bayes) |
| **Web Framework**  | Flask, HTML, CSS |
| **Model Storage**  | Joblib / Pickle |

## ⚙️ Features  
✔ Predict sentiment of any climate-related text  
✔ Trained on multi-source, credible climate datasets

✔ Easily extendable for real-time monitoring
